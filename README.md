# First Digit Finder (Java)

## 📌 Overview

This Java program finds the **first digit of a number** using a recursive approach.

---

## 🚀 How It Works

* The function repeatedly divides the number by 10.
* It continues until the number becomes a single digit.
* That single digit is the first digit of the original number.

---

## 🧠 Logic

* **Base Case:**
  If number < 10 → return number
* **Recursive Case:**
  Call function with `n / 10`

---

## 📥 Input

A positive integer (hardcoded in the program)

## 📤 Output

The first digit of the given number

---

## 💡 Example

Input:
976

Output:
First digit: 9

---

## 🛠️ Concepts Used

* Recursion
* Integer division
* Java methods

---

## 📌 Use Cases

* Digit manipulation problems
* Learning recursion basics
* Data processing tasks

---

## ▶️ How to Run

1. Compile the program:

   javac firstDigit.java
   
3. Run the program:

   java firstDigit
